# Travis-CI example for Go using Go 1.11 modules

This is an example of how a Go 1.11 module built with Travis CI.


## Local Build and Test

You can use go get command:

    go get github.com/guidoman/travis-gomod-example

Testing:

    go test github.com/guidoman/travis-gomod-example/...


## Travis-CI

You can see a build status: https://travis-ci.org/guidoman/travis-gomod-example
